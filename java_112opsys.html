1.1.2 Operating Systems

Operating System (OS) is a software program that:

  • controls peripheral devices (like manages the file system)
  • supports multitasking, by scheduling multiple programs or tasks to execute during the same interval
  • allocates memory to each program, so that there is no conflict among the memory of any programs running at the same time
  • prevents the user from damaging the system. for instance, it prevents user programs from overwriting the operating system or another program's memory

The OS loads, or boots, when the computer system is turned on and is intended to run as long as the computer is running.

Some OSs you may encounter:

  • Microsoft Windows
  • Android (mobile)
  • Linux
  • macOS
  • iOS (mobile)
